User Instructions

This webpage allows you to search for elements in the Periodic Table.

  • Voice Search: Click "Search by Speech" and say the name of an element you want to search.
  • Search Bar: Enter an element's symbol, name, atomic number, or atomic mass. Press the "Search" button to filter the results. When searching for an element name you search suggestions will pop up after you type 3 letters.

After performing a search, the page will automatically scroll to display the matching elements.
